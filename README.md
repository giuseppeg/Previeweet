# Previeweet

Previeweet is browser extension which provides Twitter users with small image previews to the right of tweets.

## HOWTO use it


Download the Chrome-extension: http://previeweet.com

## Supported Services

* Twitter (pic.twitter.com)
* Instagram
* Photobucket
* Facebook
* Apple
* Yfrog
* Twitpic
* Twitvid
* Flickr
* Imgur
