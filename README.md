# Previeweet

Previeweet is an awesome browser extension that provides Twitter users with small image previews to the right of tweets.

## HOWTO use it


Download the Chrome-extension: http://previeweet.com

## Supported Services

* Twitter (pic.twitter.com)
* Instagram
* Photobucket
* Facebook
* YouTube
* Apple
* Yfrog
* Twitpic
* Twitvid
* Flickr
* Imgur

and many more!
